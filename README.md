## ğŸš€ Project Overview

Understanding Data Structures can be difficult without visual context. **Visual DSA** bridges the gap between theoretical C++ implementations and visual understanding. 

We utilize **Emscripten** to compile our C++ code (`combined_dsa.cpp`) into a `.wasm` binary. This allows the browser to execute the exact C++ logic used in standard academic environments while rendering the results dynamically on an HTML5 Canvas.

### 1. AVL Tree Visualization

* **Automatic Balancing:** Visualizes Left (LL), Right (RR), Left-Right (LR), and Right-Left (RL) rotations.

## ğŸ’» How to Run Locally

Because this project uses WebAssembly, it requires a local web server to serve the `.wasm` MIME type correctly. You cannot simply open `index.html` by double-clicking it.

### Prerequisites

  * A modern web browser (Chrome, Firefox, Edge).
  * Python (for a quick local server) OR VS Code with "Live Server" extension.

### Steps

1.  **Clone the repository:**

    ```bash
    git clone [https://github.com/Roay-Abdullah/Data-Structure-Visualizer.git](https://github.com/Roay-Abdullah/Data-Structure-Visualizer.git)
    cd your-repo-name
    ```

2.  **Start a Local Server:**

      * **Using Python 3:**
        ```bash
        python -m http.server 8000
        ```
      * **Using VS Code:**
        Open the folder in VS Code, right-click `index.html`, and select **"Open with Live Server"**.

3.  **Access the Application:**
    Open your browser and navigate to `http://localhost:8000` (or the port provided by your server).
